프로젝트 설정 변경하기 (이름·초대 코드·데이터 저장)

프로젝트 설정 페이지에서 프로젝트 이름을 변경하고, 초대 코드를 확인·복사하고, 데이터 저장 경로·자동 삭제·주간 리포트를 관리하는 방법

프로젝트의 이름, 멤버를 부를 때 쓰는 초대 코드, 그리고 데이터를 어디에 얼마나 보관할지는 모두 프로젝트 설정 페이지에서 관리한다. 사이드바에서 [프로젝트 설정]을 눌러 진입한다.

프로젝트 이름 변경하기

  1. 프로젝트 정보 영역에서 프로젝트명 옆 [이름 변경]을 누른다.
  2. 변경 팝업에 현재 이름이 표시된다. 새 이름을 입력하고 [저장하기]를 누르면 반영된다.

초대 코드 확인·복사하기

  1. 프로젝트 초대 코드 보기를 누른다. 프로젝트 ID 앞 6자가 표시된다.
  2. 복사 아이콘을 누르면 코드가 복사되고 "프로젝트 초대 코드가 복사되었습니다." 안내가 뜬다. 초대하려는 사람에게 이 코드를 전달하면, 상대는 회원가입 시 코드를 입력해 참여를 요청할 수 있다.

데이터 저장 관리하기

  1. 저장 경로 설정: 입력폼에 저장 경로를 적고 [경로 저장]을 누른다. 경로가 실제로 존재해야 저장되며, 없으면 에러가 표시된다. 경로 저장을 누르지 않고 페이지를 벗어나면 이전 경로로 되돌아간다.
  2. 저장 공간 사용 현황: 전체 용량 대비 사용·잔여 용량이 바차트로 표시된다.
  3. 자동 삭제 기준 설정: 스위치를 켜면 두 가지 기준으로 오래된 데이터를 자동 정리한다. ① 설정한 보관 기간이 지난 데이터는 다음날 새벽 3시에 삭제되고, ② 설정한 사용 용량을 넘으면 가장 오래된 탐지 데이터부터 순차 삭제된다. 기존 탐지 데이터·안전 확인 현황에 남은 장소 정보는 유지된다.
  4. 주간 요약 리포트 발송: 켜면 매주 월요일 오전 8시에 에디터 권한 유저에게 전주 요약 리포트가 이메일로 발송된다.

설정 저장하기

  • 변경 후 설정 저장을 누르면 "프로젝트 설정이 저장되었습니다" 안내가 화면 중앙에 표시된다.
  • 저장하지 않고 페이지를 벗어나면 변경한 값이 초기화된다.
