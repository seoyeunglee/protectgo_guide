설명

어떤 탐지 유형을 골라야 하나

영상·임계치·통계·발전 효율 탐지의 차이와, 감시하려는 데이터·목적에 따라 어떤 유형을 선택할지 설명

탐지 유형은 다섯 가지지만, 고르는 기준은 단순하다. 무엇을 감시하느냐(영상인가 수치인가)와 어떻게 판정하느냐(고정된 선인가 평소 패턴인가)로 갈린다.

먼저 — 영상인가, 수치인가

  • 사람·차량·화재처럼 "보이는 것"을 감시한다면 → 영상 이상 탐지. 비전 AI가 객체를 인식한다.
  • 센서·DB의 "측정값"을 감시한다면 → 임계치 또는 통계 분석 탐지.

수치라면 — 고정된 선인가, 평소 패턴인가

상황 적합한 유형
"이 값을 넘으면 무조건 위험" — 명확한 기준선이 있다 센서 임계치 / DB 임계치 탐지
절대값은 정상이라도 "평소와 다르게 튀면" 잡고 싶다 통계 분석 탐지 (Z-Score·변화율·EWMA)
외부 DB(InfluxDB)에 이미 값이 쌓이고 있다 DB 임계치 탐지
태양광 발전소 — 예측 대비 발전량 저하를 본다 발전 효율 저하 탐지

영상 탐지 vs 통계 탐지 — 정리

항목 영상 이상 탐지 통계 분석 탐지
데이터 소스 CCTV / 듀얼 열화상 센서(수치)
탐지 단위 객체(사람·지게차·안전모) 측정값의 통계적 패턴
결과물 바운딩 박스 + 클래스명 분석값(Z-Score·변화율·EWMA)
시각화 이미지 + 영상 라인 차트 + 평균값 점선
권장 환경 카메라 3M↑, 야외 적합 안정적인 센서 데이터 수집

한 시나리오에 섞어 쓸 수 있다

탐지 유형은 배타적이지 않다. 같은 장소에서 CCTV로 작업자 안전(영상)을, 진동 센서로 설비 임계치(센서)를, 발전량으로 효율 저하(통계)를 동시에 감시할 수 있다. 노드 캔버스에서 각각의 흐름을 그려 한 시나리오에 담거나 여러 시나리오로 나눈다.
