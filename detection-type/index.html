탐지 유형별 설정

탐지 시나리오·노드 도메인이 "노드를 어떻게 조립하는가"를 다룬다면, 이 도메인은 "무엇을 어떻게 탐지하는가"를 탐지 유형별로 다룬다. 같은 노드 캔버스 위에서도 영상·임계치·통계·발전 효율은 보는 데이터와 판정 방식이 다르다.

탐지 유형 5종

유형 데이터 소스 판정 방식 알림 트리거
영상 이상 탐지 CCTV·듀얼 열화상 영상 비전 AI 모델 (객체 검출) 객체 인식 + 이벤트 조건
DB 임계치 탐지 외부 DB(InfluxDB) 수치 임계치 비교 임계치 초과
센서 임계치 탐지 진동·온도·복합가스·열화상 센서 수치 임계치 비교 임계치 초과
통계 분석 탐지 센서(다중) Z-Score / 변화율 / EWMA 통계적 이상치
발전 효율 저하 탐지 전력 센서 + 발전량 데이터 발전량 예측값 대비 실측값 예측 대비 저하 추세(EWMA)

임계치 vs 통계 vs 영상 — 무엇이 다른가

  • 임계치 탐지(DB·센서)는 "값이 선을 넘었는가"를 본다. 가장 단순하고 명확하다 — 진동이 10mm/s를 넘으면 탐지.
  • 통계 분석 탐지는 "평소 패턴에서 벗어났는가"를 본다. 고정된 선이 아니라 평균·변동폭을 기준으로 이상치를 잡는다 — 절대값은 정상 범위라도 갑자기 튀면 탐지.
  • 영상 이상 탐지는 "화면에 무엇이 보이는가"를 본다. 수치가 아니라 비전 AI가 사람·지게차·화재 같은 객체를 인식한다.
  • 발전 효율 저하 탐지는 신재생(태양광) 도메인 특화로, "예측한 만큼 발전하고 있는가"를 EWMA로 추적한다.

탐지 이후 — 확인 현황

탐지가 잡히면 끝이 아니라, 담당자가 현장을 확인하고 처리 상태를 남긴다. 모든 탐지 유형이 공통으로 미해결 → 확인중 → 해결의 확인 흐름을 따른다.

튜토리얼

처음부터 끝까지 따라 하며 익히는 단계별 안내. 전체 시나리오 제작은 [탐지 시나리오·노드](../../detection-node/tutorial/) 도메인을 참고한다. (준비 중)

하우투

  • DB 임계치 탐지 설정하기 — 외부 DB(InfluxDB)에 저장된 센서 측정값을 데이터베이스 노드로 불러와 임계치로 모니터링하는 방법
  • 발전 효율 저하 탐지 설정하기 (태양광) — 태양광 발전량 예측값 대비 실측값이 지속 부족한 추세를 EWMA로 추적해 발전 효율 저하를 탐지하도록 설정하는 방법
  • 센서 임계치 탐지 설정하기 — 진동·온도·복합가스·열화상 센서 값이 정해진 임계치를 넘으면 탐지하도록 설정하는 방법
  • 영상 이상 탐지 설정하기 (쓰러짐·안전모 등) — CCTV·열화상 영상에 비전 AI 모델로 작업자·지게차 등 객체를 탐지하도록 설정하는 방법과 탐지 데이터 확인
  • 탐지 확인 현황 보기·대응하기 — 탐지된 이상을 미해결→확인중→해결로 처리하고, 이상 여부 판정과 문제 유형을 입력해 일괄 확인하는 방법
  • 통계 분석 탐지 설정하기 (Z-Score·변화율·EWMA) — 고정 임계치 대신 평소 패턴 대비 이상치를 잡는 통계 분석 탐지(Z-Score·변화율·EWMA)를 설정하고 탐지 데이터를 읽는 방법

레퍼런스

임계치 입력 규칙·측정 항목 등 정확한 사양. 임계치 공통 규칙은 [노드 연결 규칙](../../detection-node/reference/node-connection-rules.html)도 함께 본다. (준비 중)
