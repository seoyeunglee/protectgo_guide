하우투

영상 이상 탐지 설정하기 (쓰러짐·안전모 등)

CCTV·열화상 영상에 비전 AI 모델로 작업자·지게차 등 객체를 탐지하도록 설정하는 방법과 탐지 데이터 확인

영상 이상 탐지는 CCTV나 열화상 영상에서 비전 AI가 객체를 인식해 위험을 잡는 방식이다. 작업자 쓰러짐, 안전모 미착용, 지게차 충돌 위험처럼 "화면에 무엇이 보이는가"로 판정한다.

설정 절차

  1. 시나리오에 영상 데이터 노드(CCTV·듀얼 열화상)를 놓고 데이터 소스를 연결한다 — 데이터 노드 연결하기.
  2. 감시할 영역만 보려면 탐지 영역(ROI)을 지정한다. 화면 전체를 보려면 생략한다.
  3. 영상 탐지 설정 노드(예: 쓰러짐 탐지)를 연결하고, 탐지 영역별 탐지 대상(작업자·지게차·안전모 등)을 매핑한다 — 탐지 설정 노드 구성하기.
  4. 이벤트 설정 노드로 발생 조건을 정하고, 필요하면 알림 노드를 붙인다.

권장 설치 환경: 카메라 높이 3M 이상, 야외 환경에 적합하다. 작업자 안전 모델은 쓰러짐·충돌 위험·보호구 미착용 등을 지원한다.

탐지 데이터 확인하기

탐지가 잡히면 탐지 데이터에서 무슨 일이 있었는지 본다.

  • 좌측 패널: 발생 장소·설비·탐지 설정 이름·데이터 소스명·탐지 영역명·발생일시.
  • 우측 메인: 바운딩 박스(또는 ROI)가 표시된 탐지 이미지. 이미지 다운로드·크게 보기·영상 재생을 할 수 있다.
  • 영상 재생: 이벤트 발생 전후 5초를, 바운딩 박스 없는 실영상으로 보여준다. 반복 재생되며 슬라이더로 임의 시점으로 이동할 수 있다(전체 화면은 ESC로 종료).

실시간 영상 보기

  • 실시간 데이터 탭에서 다른 카메라로 전환하며 현재 상황을 본다.
  • 15초 이상 영상이 호출되지 않으면 "데이터를 불러올 수 없음" 안내가, 외부 네트워크 접속이면 "네트워크 문제로 영상을 표시할 수 없습니다" 안내가 표시된다.
