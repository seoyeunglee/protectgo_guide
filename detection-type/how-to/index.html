탐지 유형별 설정 · 하우투

탐지 유형별 설정 절차. 영상·임계치·통계·발전 효율 탐지를 각각 어떻게 설정하는지 안내한다.

  • DB 임계치 탐지 설정하기

    외부 DB(InfluxDB)에 저장된 센서 측정값을 데이터베이스 노드로 불러와 임계치로 모니터링하는 방법

  • 발전 효율 저하 탐지 설정하기 (태양광)

    태양광 발전량 예측값 대비 실측값이 지속 부족한 추세를 EWMA로 추적해 발전 효율 저하를 탐지하도록 설정하는 방법

  • 센서 임계치 탐지 설정하기

    진동·온도·복합가스·열화상 센서 값이 정해진 임계치를 넘으면 탐지하도록 설정하는 방법

  • 영상 이상 탐지 설정하기 (쓰러짐·안전모 등)

    CCTV·열화상 영상에 비전 AI 모델로 작업자·지게차 등 객체를 탐지하도록 설정하는 방법과 탐지 데이터 확인

  • 탐지 확인 현황 보기·대응하기

    탐지된 이상을 미해결→확인중→해결로 처리하고, 이상 여부 판정과 문제 유형을 입력해 일괄 확인하는 방법

  • 통계 분석 탐지 설정하기 (Z-Score·변화율·EWMA)

    고정 임계치 대신 평소 패턴 대비 이상치를 잡는 통계 분석 탐지(Z-Score·변화율·EWMA)를 설정하고 탐지 데이터를 읽는 방법
