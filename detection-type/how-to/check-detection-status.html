탐지 확인 현황 보기·대응하기

탐지된 이상을 미해결→확인중→해결로 처리하고, 이상 여부 판정과 문제 유형을 입력해 일괄 확인하는 방법

탐지가 잡혔다고 끝이 아니다. 담당자가 현장을 확인하고 실제 문제였는지, 오탐이었는지 판정해 기록을 남겨야 한다. 이 처리 흐름은 영상·임계치·통계 등 모든 탐지 유형이 공통으로 따른다.

처리 상태 — 미해결 → 확인중 → 해결

상태 의미
미해결 탐지됐지만 아직 아무도 확인하지 않음
확인중 담당자가 "확인 시작"을 눌러 처리 중 (해당 장소 담당자에게 자동 배정)
해결 이상 여부 판정과 문제 유형 입력을 마쳐 확인 완료

"확인 시작"은 해당 장소의 담당자만 누를 수 있고, 누른 사람에게 자동으로 배정된다.

한 건씩 또는 일괄로 처리하기

여러 미해결 건을 한 번에 처리할 때는 일괄 확인 조치 패널을 쓴다.

  1. 화면 우측의 일괄 확인 패널을 연다. 미해결 탐지건만 자동으로 모여 표시된다.
  2. 처리할 건을 체크한다(전체 선택도 가능). 1건 이상 고르면 [일괄 확인]이 활성화된다.
  3. 1차 판정: 선택한 건이 실제 문제면 [이상 발생 확인], 오탐이면 [이상 없음]을 누른다.
  4. 이상 발생 확인이라면 문제 유형을 고른다. 탐지 종류별로 최소 1개 이상 선택해야 등록된다. 기본 제공 유형에서 고르거나 직접 입력(최대 20자)한다.
  5. [등록하기]로 확정하면 선택 건이 '확인 완료(해결)'로 바뀐다.

자주 쓰는 문제 유형은 "이 문제 유형을 저장하여 다음에도 사용"을 체크해 저장한다 — 해당 장소·탐지 유형의 문제 유형으로 등록되어 다음에 바로 고를 수 있다.

처리 중 다른 사람이 먼저 처리했다면

일괄 확인은 누를 때 선택 건 중 현재 "미확인" 상태인 건만 다시 검증해 처리한다. 그 사이 다른 담당자가 일부를 먼저 처리했다면 "N건 중 M건 처리완료되었습니다" 안내가 표시된다 — 중복 처리를 막기 위한 동작이다.

다음 단계

  • 모바일에서 같은 확인 흐름을 처리하는 방법은 모바일 앱(후속 도메인)의 안전 체크리스트를 참고한다.
  • 탐지 유형별 설정으로 돌아가려면 탐지 유형별 설정 소개를 본다.
