센서 임계치 탐지 설정하기

진동·온도·복합가스·열화상 센서 값이 정해진 임계치를 넘으면 탐지하도록 설정하는 방법

센서 임계치 탐지는 가장 직관적인 탐지다. 진동·온도·가스 농도 같은 측정값이 정해 둔 선(임계치)을 넘으면 이상으로 잡는다.

설정 절차

  1. 시나리오에 센서 데이터 노드(진동·온도·복합가스·열화상)를 놓고 데이터 소스를 연결한다.
  2. 임계치 설정 노드를 연결한다. 같은 센서 유형만 연결되며, 여러 개 등록할 수 있다.
  3. 센서 카드에서 측정 항목을 확인한다 — 복합 가스 같은 다중값 센서는 항목별로, 진동 같은 단일값 센서는 단일 항목으로 표시된다.
  4. 각 항목에 임계치를 입력한다. 옆의 측정 범위 안의 값을 숫자로 넣고(소수점 셋째 자리까지), "이상" 또는 "이하" 기준을 정한다.
  5. 필요 없는 항목은 활성화 체크박스로 끈다. 센서 카드의 [전체 활성화/전체 연결 해제]로 한 번에 켜고 끌 수 있다.

측정 범위를 벗어난 값이나 빈 값으로 저장하면 에러가 표시된다.

열화상 카메라의 특수 처리

열화상은 온도를 임계치로 본다. 단, 탐지 영역(ROI) 설정 여부에 따라 기준이 달라진다.

케이스 기준
탐지 영역 미설정 화면 전체의 최고/최저/평균 온도
탐지 영역 설정(ROI) ROI별 최고/최저/평균 온도

탐지 영역을 만들어 뒀더라도 데이터 노드에서 탐지 설정 노드로 직접 연결하면 "탐지 영역 미설정"으로 처리된다. 영역별로 보려면 ROI 노드를 거쳐 연결한다.
