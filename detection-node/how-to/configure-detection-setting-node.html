탐지 설정 노드 구성하기

데이터 노드에 탐지 설정 노드를 연결해 영상 탐지 대상이나 센서 임계치를 정의하는 방법과, 연결 가능한 데이터 제약

탐지 설정 노드는 데이터를 "무엇으로 볼지" 정하는 단계다. 영상이라면 탐지할 대상을, 센서라면 넘으면 안 되는 임계치를 정의한다. 데이터 노드와 이벤트 노드 사이에 놓인다.

모든 데이터 노드를 아무 탐지 설정에나 붙일 수 있는 건 아니다. 탐지 유형에 맞는 데이터만 연결되며, 맞지 않으면 연결 시점에 안내 팝업이 뜨고 차단된다. 데이터가 아직 연결되지 않은 노드도 붙일 수 없다.

영상 탐지 대상 설정하기

  1. 데이터 노드(또는 ROI 노드)에 영상 탐지 설정 노드(예: 쓰러짐 탐지)를 연결한다.
  2. 탐지 영역별로 탐지 대상을 지정한다(예: 작업자, 지게차). 영역과 탐지 객체를 매핑한다.
  3. 전체 활성화 / 전체 연결 해제 토글로 영역을 한 번에 켜고 끌 수 있다.

센서 임계치 설정하기

  1. 센서 데이터 노드에 임계치 설정 노드를 연결한다. 같은 센서 유형만 연결되며, 여러 개 등록할 수 있다.
  2. 센서 카드에서 측정 항목을 확인한다. 복합 가스처럼 다중값 센서는 항목별로, 진동처럼 단일값 센서는 단일 항목으로 표시된다.
  3. 임계치 수치를 입력한다. 옆에 표시된 센서 측정 범위(예: "500 ~ 1500") 안의 값을 숫자로 넣는다. 소수점은 셋째 자리까지 가능하다.
  4. 기준을 "이상" 또는 "이하"로 정한다 — 그 값을 넘거나 밑돌면 탐지로 잡는다.

범위를 벗어난 값이나 빈 값으로 저장하면 에러 텍스트가 표시된다. 열화상 카메라는 탐지 영역을 설정하지 않으면 화면 전체의 최고/최저/평균 온도를, 영역을 설정하면 영역별 온도를 기준으로 삼는다.

다음 단계

  • 탐지 설정만으로는 알림이 발생하지 않는다. 이벤트 설정 노드를 연결해 발생 조건을 정해야 탐지 데이터가 생긴다 — 이벤트 설정 노드 구성하기.
  • 임계치·통계 등 탐지 유형별 자세한 설정은 탐지 유형별 설정 도메인을 참고한다.
