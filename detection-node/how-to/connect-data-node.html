데이터 노드 연결하기 (센서·CCTV·열화상·DB)

시나리오 캔버스에 데이터 노드를 놓고 설치 장소의 데이터 소스를 불러와 연결하는 방법과, 영상 데이터의 탐지 영역(ROI) 설정

데이터 노드는 탐지의 입력이다. 센서 측정값이나 카메라 영상을 캔버스로 끌어들여, 뒤에 붙는 탐지 설정 노드가 분석할 재료를 공급한다.

데이터 사이드바에는 이 시나리오 장소에 등록된 데이터 유형만 나타난다. CCTV·열화상·진동·온도·복합가스처럼 유형 단위로 보이며, 구체적인 데이터 소스(어느 카메라·센서인지)는 노드 안에서 고른다.

데이터 노드 놓고 연결하기

  1. 사이드바에서 원하는 데이터 유형 노드(예: CCTV, 진동 센서)를 캔버스로 드래그한다.
  2. 노드 안의 설치 장소는 현재 시나리오 장소로 고정되어 변경할 수 없다.
  3. 데이터 소스명 셀렉박스에서 연결할 실제 소스(예: "컴프레서 진동센서 ORM1")를 고른다.
  4. [데이터 불러오기]를 누른다. 연결되면 장소·설비·측정 항목이 표시되고, 실시간 데이터 영역이 나타난다.

[데이터 불러오기] 후 15초 동안 응답이 없으면 연결 실패 화면이 뜬다. 외부 네트워크에서 접속했다면 "네트워크 문제로 영상을 표시할 수 없습니다" 안내가 표시된다 — 사내망/VPN에서 다시 시도한다.

영상 데이터에 탐지 영역(ROI) 설정하기

CCTV나 듀얼 열화상 같은 영상 데이터는 화면 전체가 아니라 관심 영역만 감시하는 게 보통이다.

  1. 영상 데이터 노드의 [탐지 영역 설정]을 눌러 탐지 영역 사이드바를 연다.
  2. 영상 위에 다각형으로 탐지 영역을 그린다. 영역을 저장하면 ROI 노드가 자동 생성되어 데이터 노드에 연결된다(별도 이름 입력은 없다).
  3. 영역이 여러 개면 각 ROI마다 뒤에서 다른 탐지 설정을 붙일 수 있다.

연결 데이터 바꾸기

  • 노드의 데이터 편집 아이콘으로 연결을 바꾸거나 해제한다. 변경 시 "데이터 변경 이후 이상 탐지가 정상 작동하지 않을 수 있습니다" 경고가 표시된다 — 이미 운영 중인 탐지에 영향을 줄 수 있으니 확인 후 진행한다.

저장하기

  • 영상 데이터는 정상 호출되면 자동 저장된다.
  • 센서 데이터는 노드의 [설정 저장] 버튼으로 전체 설정값을 저장한다.
  • 탐지 영역(ROI)은 탐지 영역 사이드바에서 별도로 저장한다.

다음 단계

  • 연결한 데이터를 무엇으로 판정할지 정하려면 탐지 설정 노드 구성하기로 이어진다.
  • 사용할 수 있는 데이터 노드 목록은 노드 카탈로그에서 확인한다.
