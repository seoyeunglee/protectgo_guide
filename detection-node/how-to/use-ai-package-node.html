AI 패키지 노드 사용하기

모델·데이터·설정을 묶은 AI 패키지 노드를 설비 또는 장소 단위로 적용해 탐지에 쓰는 방법

AI 패키지는 모델 + 필요한 데이터 소스 + 모델 설정값을 하나로 묶은 노드다. 컴프레셔 화재 예측처럼 도메인별로 검증된 묶음을 그대로 가져다 쓰면, 데이터·탐지 설정을 따로 구성하지 않아도 된다.

적용 대상 정하기 — 설비 vs 장소

AI 패키지 노드를 놓으면 먼저 적용 대상 유형을 고른다.

  • 설비: 특정 설비를 골라, 그 설비에 배포된 모델을 사용한다.
  • 장소: 장소 단위로, 프로젝트 기본 모델을 사용한다.

설비 모드로 적용하기

  1. 적용 대상에서 설비를 선택한다.
  2. 설비 셀렉박스에서 대상 설비를 고른다.
  3. 그 설비에 배포된 모델이 없으면 기본 모델 적용 안내 팝업이 뜬다 — 기본 모델로 자동 설정되며, 나중에 설비 전용 모델을 배포하면 자동으로 교체된다.
  4. 탐지 정보 영역에 모델·탐지 유형·탐지 대상이 표시된다.
  5. 데이터 소스 셀렉박스에서 해당 설비에 등록된 데이터 소스를 고르고 설정을 저장한다.

적용 대상 탭(설비/장소)을 바꿔 저장하면 이전 탭의 저장값은 초기화된다. 한 노드에는 한 대상만 적용된다.

장소 모드로 적용하기

  1. 적용 대상에서 장소를 선택한다.
  2. 프로젝트 기본 모델 정보가 자동으로 표시된다.
  3. 데이터 소스 셀렉박스에서 해당 장소에 등록된 데이터 소스를 고른다.

다음 단계

  • AI 패키지 뒤에 알림을 붙이려면 먼저 하위 속성마다 이벤트 노드를 설정해야 한다 — 이벤트 설정 노드 구성하기를 참고한다.
  • AI 패키지가 무엇을 묶은 것인지 개념은 데이터·모델 관리 도메인(후속)에서 더 다룬다.
