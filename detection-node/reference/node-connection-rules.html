레퍼런스

노드 연결 규칙·단축키

노드 간 연결 제약, 시나리오 캔버스 단축키, 저장 실패·통신 오류 시 동작을 정리한 레퍼런스

시나리오 캔버스에서 어떤 노드끼리 연결되는지, 단축키는 무엇인지, 저장이 실패하면 어떻게 되는지 정리한 레퍼런스다. 실제 구성 절차는 첫 탐지 시나리오 만들기를 참고한다.

연결 제약

연결 규칙
데이터 → 탐지 설정 탐지 유형에 맞는 데이터만 연결. 맞지 않으면 "연결 가능 노드 안내" 팝업
데이터 미연결 노드 데이터가 연결되지 않은 노드는 연결 불가 — "데이터 연결 후 다시 연결" 팝업
임계치 설정 노드 동일 센서 유형만 연결(복수 등록 가능)
탐지 설정/AI 패키지 → 이벤트 이벤트 노드는 탐지 설정/AI 패키지와 연결되어야 함
이벤트 → 알림 이벤트 노드 설정이 완료되어야 알림 노드 연결 가능(AI 패키지는 하위 속성까지)
엣지 디바이스 → 이벤트 엣지 디바이스 노드 뒤에는 이벤트 설정 노드 N개 연결 가능

단축키

동작 Windows Mac
선택 항목 삭제 Delete / Backspace Delete / Backspace
실행 취소(Undo) Ctrl+Z Cmd+Z
다시 실행(Redo) Ctrl+Shift+Z Cmd+Shift+Z

노드 내부 텍스트 입력 폼에 커서가 있으면 Delete가 노드가 아니라 텍스트만 지운다. [초기화]로 전체를 지운 뒤에도 Undo로 복구할 수 있다.

저장·통신 실패 시 동작

상황 동작
데이터 저장 실패 2초 간격 3회 재시도(총 6초). "설정값을 저장중 입니다." → 최종 실패 시 "일시적인 오류로 인해 저장에 실패했습니다. 잠시후 다시 시도해주세요"
서버 통신 오류 동일하게 3회 재시도. "설정 적용중입니다." → 최종 실패 시 "네트워크 오류로 인해 설정 적용에 실패했습니다. 문제가 지속될 경우 고객센터에 문의해 주세요"

뒤로가기 시 안내

  • 탐지 설정이 완료되지 않은 시나리오가 있으면: "설정 완료 되지 않은 탐지 시나리오가 있습니다…" 안내 팝업.
  • 저장되지 않은 노드·탐지 영역이 있으면: "저장되지 않은 설정값이 있습니다. 현재 페이지를 벗어날 시 변경 내용은 초기화됩니다…" → [전체 저장] / [뒤로 가기].
