# 노드 카탈로그 레퍼런스

시나리오 캔버스의 노드 카테고리 사이드바에서 사용할 수 있는 노드 목록이다. 노드의 역할과 연결 흐름은 [탐지 시나리오·노드 소개](../index.html)를, 실제 구성 절차는 [첫 탐지 시나리오 만들기](../tutorial/create-first-scenario.html)를 참고한다.

> 카탈로그는 등록된 모델·디바이스에 따라 늘어난다. 아래 목록은 정책 문서 기준이며, 프로젝트에 따라 표시 항목이 다를 수 있다.

## 데이터 노드

센서·카메라 입력을 연결한다.

- CCTV
- 복합가스 센서
- 열화상 카메라 싱글
- 열화상 카메라(듀얼) — 영상
- 열화상 카메라(듀얼) — 열화상
- 진동 센서
- 온도 센서

## AI 패키지 노드

모델·데이터·설정을 묶은 탐지 단위다.

- 컴프레셔 화재 발생 예측 모델
- 컴프레셔 열화상 온도 이상치 탐지
- 컴프레셔 진동 센서 이상치 탐지

## 탐지 설정 노드 (3개 카테고리)

| 카테고리 | 노드 |
|---|---|
| 작업자 안전 | 쓰러짐 탐지, 지게차 안전모 미착용 탐지, 지게차 도보 구역 침범 탐지 |
| 화재 | 영상 화재 탐지, 컴프레셔 화재 탐지 |
| 센서 임계치 | 진동 센서 임계치 탐지, 열화상 카메라 임계치 탐지, 복합 가스 센서 임계치 탐지 |

## 이벤트 설정 노드

- 개별 이벤트 설정 — 단일 소스의 발생 조건
- 융합 이벤트 설정 — 복수 소스를 하나의 이벤트로 묶음

입력값·검증 규칙은 [이벤트 설정 노드 구성하기](../how-to/configure-event-setting-node.html)에서 다룬다.

## 알림 및 제어 노드

- 대시보드 알림 설정
- 모바일 알림 설정
- 외부 스피커 알림 설정
- 담당자 자동 배정 알림 (SOP 알림)

## 노드 공통 사양

| 항목 | 동작 |
|---|---|
| 노드명 기본값 | "{기본 노드명} + 순서 숫자" (예: 열화상 카메라 싱글1). 중간 번호를 지워도 마지막 번호에 +1로 이어진다 |
| 노드명 변경 | 헤더의 이름 변경 아이콘으로 인라인 편집. 중복 이름은 불가 |
| 최소/최대화 | 헤더 아이콘으로 노드를 접고 펼친다 |
| 삭제 | 확인 팝업 "노드를 삭제하시겠습니까? 현재 노드의 설정된 값은 저장되지 않습니다." 후 [삭제 하기]로 확정 — 설정값은 복구되지 않는다 |
