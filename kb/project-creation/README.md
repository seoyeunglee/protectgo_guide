# 도메인: 프로젝트 생성 (`project-creation`)

2주 범위의 핵심 도메인 1. 프로젝트 생성·선택·멤버 초대·사용 용도 선택 등 "워크스페이스를 만들고 들어가는" 흐름을 다룬다.
출발점: `protect-go-knowledge/03-project-lifecycle.md`(기존 KB) + 화면설계서 PDF 정책 추출분(`kb/_sources/policy/`) + 프론트엔드 코드.

## Diátaxis 4유형
- [tutorial/](./tutorial/) — 학습 지향: 처음 만드는 사용자가 따라 하며 배우는 단계별 안내
- [how-to/](./how-to/) — 작업 지향: 특정 목표(예: 멤버 초대)를 달성하는 절차
- [reference/](./reference/) — 정보 지향: 정확한 사양·옵션·제한값

엔트리는 3단계(PRD 플로우로 IA·우선순위 확정) 이후 채운다. 스키마는 [schema/kb-entry.md](../../schema/kb-entry.md) 기준.
