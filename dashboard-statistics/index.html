홈 / 대시보드·통계

대시보드·통계

탐지가 만들어 낸 결과를 보는 화면이다. 대시보드는 위젯을 모아 현재 상황을 한눈에 보여주고, 통계 페이지는 기간을 두고 추이를 분석한다.

대시보드 — 위젯으로 구성하는 모니터링 화면

대시보드(GNB 첫 메뉴)는 위젯 그리드 기반이다. 에디터는 편집 모드로 위젯을 추가·이동·삭제해 화면을 직접 구성한다(매니저·뷰어는 조회).

위젯은 두 갈래로 나뉜다.

갈래 무엇을 보여주나 위젯(예)
Chart 시간·장소·유형별 추이를 차트로 시간대별 이상 발생 건수, 장소별 이상 발생 건수, 실시간 센서 변동, 탐지 발생 추이, 유형별 탐지 비율
Monitor 현재 현황·목록·도면을 이상 탐지 발생 요약, 실시간 이상 탐지 내역, 배치도, 탐지 확인 현황

여기에 스마트워치와 연동하는 작업자 상태 모니터링 위젯이 추가된다.

배치도 — 도면 위에서 보는 현황

배치도 위젯은 현장 도면 이미지 위에 비콘으로 장소를 매핑해, 어디서 이상이 발생했는지 공간적으로 보여준다. 비콘을 클릭하면 그 장소의 상세(탐지 이력·데이터 소스 현황)로 진입한다.

통계 페이지 — 기간을 두고 분석

통계 페이지(GNB 3번째)는 발생 추이·장소별 발생·유형별 발생을 기간(7일~12개월)으로 분석한다. 대시보드가 "지금"이라면 통계는 "그동안"을 본다.

튜토리얼

  • 대시보드 살펴보기 — 로그인 후 처음 만나는 대시보드의 위젯·배치도·전체 화면 보기와 이상 발생 알림 팝업을 둘러보는 과정

하우투

  • 위젯 추가·편집하기 (편집 모드) — 편집 모드로 들어가 위젯을 추가하고, 위치·크기를 조정하고, 삭제한 뒤 저장하는 방법
  • 작업자 상태 모니터링하기 (스마트워치 연동) — 스마트워치와 연동된 작업자 상태 모니터링 위젯으로 위험·주의·정상 인원과 이슈 피드를 확인하고, 작업자 리스트로 상세를 보는 방법
  • 통계 페이지에서 이상 추이 분석하기 — 통계 페이지에서 발생 추이·장소별·유형별 탭과 기간 옵션으로 이상 탐지 추이를 분석하는 방법

레퍼런스

  • 위젯 카탈로그 — 대시보드에 추가할 수 있는 위젯(Chart 5종·Monitor 4종 + 작업자 모니터링)과 기본 크기·용도를 정리한 레퍼런스

설명

  • 위젯 수치는 어떻게 계산되나 — 실시간 센서 변동 모니터링의 산출식, 비율·숫자 표기 규칙, 라인 차트 Y축 자동 산출 등 위젯이 보여주는 수치의 계산 방식을 설명
