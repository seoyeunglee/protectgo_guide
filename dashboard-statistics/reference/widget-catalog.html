위젯 카탈로그

대시보드에 추가할 수 있는 위젯(Chart 5종·Monitor 4종 + 작업자 모니터링)과 기본 크기·용도를 정리한 레퍼런스

대시보드 편집 모드의 위젯 리스트에서 추가할 수 있는 위젯이다. 추가·배치 방법은 위젯 추가·편집하기를 참고한다.

Chart — 추이·비율을 차트로 (기본 5×4)

위젯 용도
시간대별 이상 발생 건수 날짜·시간대별 탐지 추이
장소별 이상 발생 건수 장소별 발생률 (바차트)
실시간 센서 변동 모니터링 센서 평균값 대비 현재값 비교
탐지 발생 추이 기간 내 발생 변화 추이
유형별 탐지 비율 유형별 건수·전체 대비 비율

Monitor — 현황·목록·도면을

위젯 기본 크기 용도
이상 탐지 발생 요약 5×2 당일 탐지 유형을 건수 많은 순으로
실시간 이상 탐지 내역 6×4 당일 발생 이상 탐지 현황·세부
배치도 5×5 도면 + 비콘으로 장소별 탐지 현황
탐지 확인 현황 6×4 당일 이벤트의 안전 확인 현황

추가 위젯

위젯 기본 크기 용도
작업자 상태 모니터링 5×4 (최대 20×12) 스마트워치 연동, 위험/주의/정상 집계 + 이슈 피드

배치도 위젯 참고

  • 도면 이미지는 .jpg·.jpeg·.png, 10MB 이하로 업로드한다.
  • 비콘은 한 탭(배치도)에 20개 미만까지 등록할 수 있다(20개 이상 시 추가 비활성).
  • 배치도 더보기 버튼은 모든 권한에 표시되지만, [위젯 설정]은 에디터에게만 보인다.
