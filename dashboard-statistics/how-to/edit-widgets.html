위젯 추가·편집하기 (편집 모드)

편집 모드로 들어가 위젯을 추가하고, 위치·크기를 조정하고, 삭제한 뒤 저장하는 방법

대시보드는 현장에 맞게 직접 구성한다. 위젯을 더하고 배치를 바꾸는 작업은 모두 편집 모드에서 한다. 편집 모드는 에디터 권한에만 보인다.

편집 모드 들어가기

  1. 대시보드 헤더의 편집 모드 토글을 누른다.
  2. 헤더 아래에 [저장하기]·[위젯 생성] 버튼이 나타나고, 사이드바가 축소되며 그리드가 보인다.

위젯 추가하기

  1. [위젯 생성]을 누른다. 위젯 리스트 팝업이 열린다.
  2. 전체/차트/모니터링 탭에서 원하는 위젯 카드(이름·기본 크기·설명)를 고른다.
  3. 선택하면 최소 크기로 화면에 생성되어, 빈 공간 중 기존 위젯과 가까운 곳에 자동 배치된다.

남은 공간이 위젯 최소 크기보다 작으면 "위젯을 생성 하려면 위젯 최소 크기만큼의 영역을 만들어 주세요" 안내가 뜬다 — 다른 위젯을 줄여 공간을 먼저 만든다.

위치·크기 바꾸기

  • 이동: 위젯 헤더(붉은 점선 영역)를 잡고 드래그한다.
  • 크기: 오른쪽 아래 모서리에 마우스를 올리면 크기 조정 아이콘이 나타난다. 드래그로 키우거나 줄인다.
  • 다른 위젯과 겹치면 붉은 고스트박스가 표시되고, 놓으면 원래 위치·크기로 되돌아간다.

위젯 이름·기간 설정하기

  • 위젯 더보기 → [위젯 설정]에서 이름(최대 20자)과 조회 기간을 바꾼다.
  • 차트형 위젯은 [위젯 설정]과 별개로 차트 설정 팝업에서 기간·장소·항목을 더 세밀하게 설정한다.

삭제하고 저장하기

  1. 위젯 더보기 → [위젯 삭제]. 설정값이 있으면 확인 팝업이 뜬다.
  2. 편집을 마치면 [저장하기]를 누른다. "대시보드 설정이 저장되었습니다" 토스트 후 조회 모드로 돌아간다.

저장하지 않고 나가면 변경이 반영되지 않는다(삭제도 롤백). 변경사항이 있는데 편집을 종료하거나 다른 메뉴로 이동하면 확인 팝업이 뜬다.

다음 단계

  • 추가할 수 있는 위젯 전체 목록은 위젯 카탈로그를 본다.
  • 작업자 모니터링 위젯 사용법은 작업자 상태 모니터링하기로 이어진다.
