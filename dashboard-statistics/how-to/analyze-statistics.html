하우투

통계 페이지에서 이상 추이 분석하기

통계 페이지에서 발생 추이·장소별·유형별 탭과 기간 옵션으로 이상 탐지 추이를 분석하는 방법

대시보드가 "지금"을 본다면, 통계 페이지(GNB 3번째)는 기간을 두고 추이를 본다. 언제·어디서·어떤 유형의 이상이 많았는지 분석해 대응의 우선순위를 정할 때 쓴다.

세 가지 관점으로 보기

탭 무엇을 보나
발생 추이 시간 기반으로 이상 탐지가 늘고 줄어든 추이
장소별 발생 어느 장소에서 많이 발생했는지
유형별 발생 어떤 탐지 유형이 많았는지 비율

분석 절차

  1. 통계 페이지에서 분석할 탭(발생 추이·장소별·유형별)을 고른다.
  2. 차트 기간을 정한다 — 7일 / 14일 / 30일 / 6개월 / 12개월 / 직접 설정. 기간은 최소 7일 이상부터 선택할 수 있다.
  3. 직접 설정한다면 시작일·종료일을 고른다. 시작일이 종료일보다 늦으면 "조회 시작 기간을 조회 끝 기간보다 이전으로 선택해주세요" 안내가 뜬다.
  4. 차트에서 추이를 읽는다. 표시되는 수치가 소수면 소수점 둘째 자리까지만 보인다.

모바일에서는

모바일 통계는 더 단순하게 개편됐다 — 차트는 좌우 스크롤 대신 탭으로, 카드 추가 데이터는 전체보기 버튼으로, 기간 선택은 바텀시트 한 번으로, 정렬은 버튼을 누르면 최신순↔오래된순으로 즉시 전환된다(전환 시 토스트로 알림).

다음 단계

  • 대시보드에서 같은 차트를 위젯으로 보려면 위젯 카탈로그를 본다.
