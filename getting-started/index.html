시작하기

Protect Go(정식 명칭 Protect GO AI)는 제조·건설 현장의 작업자·설비·환경 데이터를 한곳에서 모아 이상을 탐지하고, 담당자에게 알림과 대응 흐름을 제공하는 스마트 공장 관리 솔루션이다. 공급사는 주식회사 아이디비(IDB)이며, 웹(브라우저)과 모바일 앱(별도 설치)으로 제공된다.

이 도메인은 제품을 처음 쓰는 사용자가 거치는 길 — 가입하고, 로그인하고, 화면을 익히고, 막혔을 때 대처하는 법 — 을 다룬다.

무엇을 할 수 있는 제품인가

  • 노드 기반 탐지 시나리오: 데이터 → AI → 이벤트 → 알림을 블록(노드)으로 연결해 "무엇을 어떻게 감지할지"를 시각적으로 구성한다.
  • 다중 데이터 통합: CCTV·열화상·진동·온도·복합가스 센서, InfluxDB 등을 한 프로젝트로 모은다.
  • 다채널 알림: 대시보드 팝업, 모바일 푸시, 외부 스피커, 담당자 자동 배정으로 이상을 알린다.
  • 현장 대응 추적: 미해결 → 확인중 → 해결의 안전 체크리스트로 처리 과정을 남긴다.

권한에 따라 보이는 화면이 다르다

로그인한 사용자의 권한에 따라 왼쪽 사이드바(GNB)에 보이는 메뉴가 달라진다.

권한 대상 사이드바
에디터 고객사 관리자 전체 메뉴
매니저 현장 담당자·실무자 대시보드 + 통계
뷰어 일반 조회자 대시보드만
어드민 공급사 IDB 내부 운영자 별도 어드민 영역

권한은 프로젝트 단위로 부여되므로, 같은 사람이라도 프로젝트마다 다른 권한을 가질 수 있다.

가입에서 첫 화면까지

  1. 회원가입: 약관 동의 → 이메일 인증 → 비밀번호 설정 → 유저 정보 입력 → 가입 완료.
  2. 로그인: 이메일·비밀번호로 접속한다. 소속 프로젝트 수에 따라 첫 화면이 갈린다 — 0개는 초대 안내, 1개는 바로 대시보드, 2개 이상은 프로젝트 선택 페이지.
  3. 메인 레이아웃: 헤더(로고·프로젝트명·알림 벨·프로필)와 왼쪽 사이드바로 모든 기능에 접근한다.

튜토리얼

  • 회원가입하고 처음 로그인하기 — 약관 동의·이메일 인증·비밀번호 설정을 거쳐 계정을 만들고, 로그인해 첫 화면(프로젝트 선택 또는 초대 안내)에 도달하는 과정

하우투

  • 내 계정·비밀번호 관리하기 — 헤더 프로필에서 개인 설정을 열어 내 정보를 확인하고, 비밀번호를 재설정하고, 로그아웃하는 방법

레퍼런스

  • 화면 구성 한눈에 보기 (헤더·사이드바) — 웹 메인 레이아웃의 헤더 요소와 왼쪽 사이드바(GNB) 메뉴 9종, 권한별 표시 범위를 정리한 레퍼런스

설명

  • 접근 제한·에러 화면이 뜨는 경우 — 외부 네트워크 차단·프로젝트 비활성화·접속 실패·에러 페이지 등 접근이 막히는 상황과 그 이유, 대처 방법을 설명
