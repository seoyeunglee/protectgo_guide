내 계정·비밀번호 관리하기

헤더 프로필에서 개인 설정을 열어 내 정보를 확인하고, 비밀번호를 재설정하고, 로그아웃하는 방법

로그인한 뒤 내 정보를 확인하거나 비밀번호를 바꾸고 싶을 때 사용한다. 진입점은 화면 오른쪽 위 프로필 아이콘이다.

내 정보 확인하고 설정 열기

  1. 헤더 오른쪽 위 프로필 아이콘을 누른다. 유저명과 아이디(이메일)가 표시된 계정 관리 팝업이 열린다.
  2. [설정]을 누르면 개인 설정 팝업으로 이동해 내 정보를 확인할 수 있다.

비밀번호 재설정하기

  1. 로그인 화면 또는 개인 설정에서 비밀번호 찾기/재설정을 시작한다.
  2. 가입할 때 쓴 이메일로 인증번호를 받아 입력한다.
  3. 새 비밀번호를 입력한다. 영문 대/소문자·숫자·특수문자를 포함해 8~20자로 만들고, 확인란에 같은 값을 한 번 더 입력한다.
  4. 변경을 완료하면 자동으로 로그아웃된다. 보안을 위한 동작이며, 변경한 비밀번호로 다시 로그인한다.

로그아웃하기

  1. 헤더 프로필 아이콘 → [로그아웃]을 누른다.
  2. "로그아웃하시겠습니까?" 확인 팝업에서 확인을 누르면 로그아웃된다. 다시 이용하려면 재로그인이 필요하다.

알아두기

  • 권한·소속·담당 장소는 본인이 직접 바꿀 수 없다. 변경이 필요하면 프로젝트의 에디터(또는 어드민)에게 요청한다 — 이들은 [계정 관리]에서 계정별로 권한·소속·담당 장소를 수정할 수 있다.
  • 모바일 알림용 앱 토큰은 계정에 종속된다. 같은 휴대폰에 다른 계정으로 로그인하면 이전 토큰은 삭제되고 새 계정으로 재발급된다.
