레퍼런스

화면 구성 한눈에 보기 (헤더·사이드바)

웹 메인 레이아웃의 헤더 요소와 왼쪽 사이드바(GNB) 메뉴 9종, 권한별 표시 범위를 정리한 레퍼런스

웹 화면은 어디서든 상단 헤더 + 왼쪽 사이드바(GNB) + 본문으로 구성된다. 처음 접속했을 때 어디에 무엇이 있는지 빠르게 익히는 데 쓴다.

헤더 (상단 고정)

위치 요소 동작
좌측 로고 클릭 시 대시보드로 이동
좌측 프로젝트명 현재 접속 중인 프로젝트. 클릭 시 다른 프로젝트로 전환하는 셀렉박스
우측 알림 벨 클릭 시 이상 탐지 알림 허브(우측 사이드바). 미읽음 알림 묶음이 있으면 빨간 배지
우측 프로필 아이콘 클릭 시 프로필 사이드바 (내 정보·설정·로그아웃)

알림 벨 배지는 미읽음 알림 묶음 개수를 표시하며, 0건이면 숨겨지고 100건 이상이면 "99+"로 표기된다.

사이드바 (GNB) 메뉴

위에서부터 순서대로:

# 메뉴 용도
1 대시보드 위젯 기반 메인 모니터링 화면
2 탐지 설정 탐지 시나리오 노드 캔버스
3 통계 발생 추이·장소별·유형별 차트
4 연결 데이터 관리 센서·DB·엣지 디바이스 연동
5 프로젝트 설정 장소·설비·데이터 저장 등 프로젝트 메타 관리
6 문서 관리 안전·설비 매뉴얼 PDF 저장(벡터 DB)
7 모델 관리 AI 모델 버전·배포 대상 관리
8 계정 관리 멤버·초대·소속·권한 관리

좌측 하단에는 문의하기, 사용자 취급 설명서 다운로드, 사이드바 확대/축소 아이콘이 있다.

권한에 따라 보이는 메뉴가 다르다

권한 사이드바에 보이는 것
에디터 전체 메뉴
매니저 대시보드 + 통계
뷰어 대시보드만
어드민 공급사 IDB 내부 어드민 영역(일반 메뉴와 별도)

매니저·뷰어로 로그인했는데 일부 메뉴가 보이지 않는다면 정상이다. 권한 차이 때문이며, 변경이 필요하면 에디터에게 요청한다.

알아두기

  • 사이드바를 축소하면 본문이 넓어진다. 축소 상태에서 아이콘에 마우스를 올리면 메뉴 이름이 툴팁으로 표시된다.
  • 현재 보고 있는 페이지의 메뉴는 사이드바에서 강조 표시된다.
