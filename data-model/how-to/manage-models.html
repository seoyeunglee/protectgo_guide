모델 등록·배포하기

모델 관리에서 모델의 기본 정보·버전 이력을 확인하고, 기본 적용 버전과 설비·디바이스별 배포 대상을 관리하는 방법

탐지가 무엇으로 판정할지는 AI 모델이 정한다. 모델 관리(GNB 7번째)에서 모델의 정보·버전을 확인하고, 어느 설비·디바이스에 어떤 버전을 적용할지 배포를 관리한다. 편집은 에디터 권한이다.

모델 정보·버전 확인하기

  1. 좌측 리스트에서 카테고리(작업자 안전·화재 등) 아래의 모델을 고른다.
  2. 기본 정보 탭에서 모델 설명·동작·탐지 항목·사용 가능한 탐지 유형·권장 환경을 본다.
  3. 버전 이력 탭에서 버전별 성능 지표(mAP)와 생성일을 확인한다.

mAP(mean Average Precision)는 객체 검출 모델이 얼마나 정확하게 탐지·분류하는지를 나타내는 지표다. 버전마다 버전 노트(300자)를 [편집]으로 남길 수 있다.

배포 대상 정하기

모델 배포 관리 탭에서 어디에 적용할지 관리한다.

  1. 기본 적용 버전을 정한다 — 특정 버전을 지정하지 않은 모든 설비·디바이스에 자동 배포되는 버전이다.
  2. 특정 대상에 다른 버전을 쓰려면, 검색창에 설비명·디바이스명을 입력해 찾아 [추가]한다.
  3. 배포 대상 테이블에서 대상별 적용 버전을 셀렉박스로 지정한다(유형 설비/엣지, 위치 정보 함께 표시).

배포 통계에서 연결된 설비·디바이스 수, 실행 중인 시나리오 수, 최근 오탐 신고 건수를 함께 본다.

알아두기 — 모델 우선순위

같은 탐지라도 가장 작은 단위에 배포된 모델이 우선 적용된다.

엣지 디바이스  >  설비  >  프로젝트

설비에 전용 버전을 배포하면 기본 적용 버전보다 그 버전이 먼저 쓰이고, 엣지 디바이스에 배포된 모델은 그보다 더 우선한다. 개념은 AI 패키지란? 모델은 어떤 순서로 쓰이나에서 더 다룬다.
